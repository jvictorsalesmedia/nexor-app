Política de Privacidade

Última atualização: julho de 2026

Esta política explica como o Nexor (app de organização, rotina, projetos, financeiro e produtividade, incluindo o assistente Nex AI) coleta, usa e protege as informações dos usuários.

Quais dados coletamos

  • Dados de cadastro: nome do responsável, nome do negócio, e-mail, número de WhatsApp e foto, informados no pré-cadastro.
  • Dados operacionais que você mesmo cadastra no uso do app: tarefas, projetos, leads, clientes, lançamentos financeiros, hábitos e equipe.
  • Mensagens enviadas ao Nex AI (pelo site ou, quando habilitado, pelo WhatsApp), para interpretar o pedido e criar registros, responder perguntas ou gerar relatórios.

Como usamos os dados

Os dados são usados exclusivamente para operar o Nexor: organizar as informações do seu negócio, permitir login e controle de acesso, e permitir que o Nex AI interprete pedidos em texto (via API do Google Gemini) para criar tarefas, lançamentos, leads, hábitos, responder perguntas sobre seus próprios dados ou gerar relatórios em PDF.

Quando o Nex AI processa uma mensagem (seja digitada no site ou recebida por WhatsApp), o texto da mensagem é enviado à API do Google Gemini apenas para interpretação — não é usado para treinar modelos de terceiros nem compartilhado para qualquer outra finalidade.

Armazenamento

Os dados ficam armazenados em banco de dados gerenciado (Supabase/PostgreSQL), com controle de acesso por login e senha. Senhas nunca são armazenadas em texto simples pela equipe do Nexor.

Compartilhamento com terceiros

Não vendemos nem compartilhamos seus dados com terceiros para fins comerciais. Os únicos serviços de terceiros envolvidos no funcionamento do Nexor são: Supabase (banco de dados e autenticação), Google Gemini (interpretação de texto do Nex AI) e, quando habilitado, a API do WhatsApp Business (Meta) para envio e recebimento de mensagens.
